# 流程配置项


- 为每个流程设立一个配置项
- 日常用户只修改 excel 或者 json 即可，不需要直接修改流程脚本代码


## 配置项为 excel 格式

建立一个excel文件，如：配置项.xlsx

参考示例： Excel读写


## 配置项为 Json 格式

建立一个 Json 文件，如：配置项.json

```javascript
global.global_config = require('./配置项.json')
```


## 动态流程配置

流程执行时，可手动输入流程配置项目，输入后流程继续执行。

API：用户输入 waitInput

参考示例： 用户手动输入变量示例.js
