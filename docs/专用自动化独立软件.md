## Hardware Device Support

The pbottle host computer software can communicate with hardware peripherals (lower-level computers) through:

1. Serial port, CAN bus, USB, Ethernet, Bluetooth, WiFi, MQTT, etc.
2. PLC device manufacturer-specific communication protocols: Omron Fins, Mitsubishi MC, Siemens S7, Modbus, OPC, etc.
3. Custom or dedicated hardware SDK development mode for peripheral data communication and control.
   
Other device types:

- Industrial camera drivers
- Motion controller drivers
- PLC lower-level computer drivers
- Sensor drivers
- Data acquisition cards
- Custom peripheral hardware driver SDK
