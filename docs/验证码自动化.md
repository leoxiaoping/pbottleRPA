# 验证码自动化

**温馨提示：小瓶RPA本身以模拟操作和数据处理为主，它不具有任何破解功能**

优先选择记住登录状态、U盾、定时刷新等方法保持用户权限的会话。

## 简单图形验证码
   
小瓶RPA文字OCR模块可以解决简单字母数字验证码

## 人工辅助半自动化

流程如下：

小瓶RPA获取验证码截图 -》发送系统管理员手机，人工识别 -》识别结果发送回小瓶RPA自动输入-》完成验证！！

一般系统都有记住登录状态功能，请优先勾选，一台电脑只需登录一次。
