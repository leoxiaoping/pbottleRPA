## 自动流程入门示例

```javascript
const pbottleRPA = require('./pbottleRPA')  //引入小瓶RPA nodejs模块

pbottleRPA.openURL('https://www.baidu.com/') // 用浏览器打开百度
pbottleRPA.paste('小瓶RPA官网')  //输入搜索词
pbottleRPA.keyTap('enter')  //确认搜索
```


