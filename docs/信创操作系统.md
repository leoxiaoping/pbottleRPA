# Using pbottleRPA on Domestic Operating Systems

pbottleRPA supports domestic operating systems from V2024.8, with full native support for all functional modules.


![pbottleRPA Domestic OS Version](https://www.pbottle.com/static/upload/20241020/17294114803702.png)

## Supported Operating Systems

1. KylinOS (Galaxy Kylin)
2. UOS (Unified Operating System)
3. Ubuntu OS
   
   ![Kylin Certification Certificate](https://www.pbottle.com/static/upload/20241204/17332947879554.jpg)
   ![UOS Certification Certificate](https://www.pbottle.com/static/upload/20241022/17295659104480.jpg)

## Install pbottleRPA

Search for 'pbottleRPA' in the official app store.

(KylinOS + UOS)


## Domestic OS Platform Differences

- The domestic OS version of pbottleRPA has **exactly the same functional modules** as the Windows version.
- You can use the Windows version to test flow scripts.
- The free personal download version supports x86_64 CPU architecture.
- The Enterprise Edition supports ARM, LoongArch, and RISC-V architectures.
