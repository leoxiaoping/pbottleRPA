#  小瓶RPA

#### 介绍
针对中小企业、职场达人的RPA自动化流程软件。
像挂QQ一样使用简单的RPA软件。 **本软件将长期更新请放心使用。**

#### 安装教程

1.  下载exe运行基座  pbottleRPA.zip  [基座exe程序绿色版](https://gitee.com/pbottle/pbottle-rpa/releases)
2.  电脑管家 软件中心安装 nodejs
3.  运行基座选择脚本即可开始运行
